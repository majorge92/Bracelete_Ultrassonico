# 								Bracelete Ultrassônico :man_technologist:

### Projeto da disciplina de Laboratório de Software e Projetos do Curso de Ciência da Computação da UNIFG.

O protótipo tem o objetivo de facilitar a vida dessas pessoas desenvolvendo um adereço que é capaz de detectar obstáculos, utilizado no braço o usuário deficiente visual terá como saber se há algum obstáculo próximo a ele, através do bracelete ultrassônico.

A linguagem de programação utilizada para desenvolver o protótipo foi C.

## Softwares

* Arduino IDE
* Visual Stúdio
* Linguagem de Programação C
* Tinkercad(Web)

### Hardwares

* Arduino
* Hc Sr04 - Sensor Ultrassônico
* Motor de Vibração
* Buzzer
